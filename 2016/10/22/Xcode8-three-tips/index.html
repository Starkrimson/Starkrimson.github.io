Add Documentation

接下来要分享的是添加注释文档的快捷键： ⌘ command + ⌥ option + /

这个快捷键的功能跟 VVDocumenter 一样，可以给函数名，变量名等生成好看的注释文档

在调用函数的时候，按下 ⌥ option 和左键，就能看到该函数的注释信息

噢对了，上面提到到两个 Literal 并不支持 OC (反正我是敲不出来了) 。注释文档这个快捷键都支持，Swift 和 OC 都可以用
