ReadOn 1.4 发布！性能优化，提高用户体验。

Starkrimson
2017-11-02 (Updated: 2022-04-07)
ReadOn Release

ReadOn 1.4 已发布。

ReadOn 1.4 发布！性能优化，提高用户体验。

2017-11-02

  • 优化界面逻辑，页面滑动更流畅

ReadOn 1.3.0 发布！更换主题，切换字体

2017-10-28

  • 暗色主题

    在设置中可以一键切换主题，未来将加入更多颜色主题。

  • 增强阅读外观选项

    现在阅读外观选项不止能够调整字体大小，还能切换字体，选择主题。

    but，小抱歉上面提到的功能需要内购升级。

  • 单元格侧滑手势

    现在向左侧滑单元格，就能呼出已读、收藏和分享等常用菜单按钮。

  • 移除所有广告

    App 内所有广告都已移除。

ReadOn 1.2.3 发布！适配 iPhone X

2017-10-17

  • 适配 iPhone X

    为即将到来的 iPhone X 做准备。

  • 增加设置项

    增加了多个设置项，包括「标记全部已读需要确认」、「隐藏已读的文章」和「启动时更新文章」。

  • 修复上个版本「标记已读」无效 bug

    上个版本写错了一个参数，导致「标记已读」失效或者导致 app crash。1.2.3 版本将修复该功能。

ReadOn 1.2.2 发布！统一管理源

2017-10-12

  • 管理订阅源

    在「管理」界面，可以查看最近添加或移除的源；统一移动源到其他文件夹或者移除它们。

  • 调整阅读字体大小

    在阅读界面，点击「阅读外观选项」，可以调整字体大小。该功能需要内购升级。

  • 去除广告

    同样包含在升级选项，升级后可以移除 app 内的广告，获得纯粹的阅读体验。

ReadOn 1.2.1 发布！VoiceOver 下体验优化

2017-09-23

  • VoiceOver 下体验优化

    以前忽视了 VoiceOver，根据 App Stroe 的留言反馈，我体验了下使用 VoiceOver 操作 ReadOn ，确实不是很好。这个版本优化了很多这方面的体验，相信可以尽可能地无障碍使用 ReadOn 阅读。

  • 搜索更多的源

    现在可以搜索到更多的源了，帮你发现更多有趣的内容。

  • 增加右侧栏

    😄全新的设计，还在摸索中，目前右侧栏用于投放广告。可能会引起一些用户的反感，还是希望大家能支持一下独立开发者。

ReadOn 1.2 发布！新增同步订阅源功能

2017-09-05

  • 一个神奇的按钮

    「设置」增加了一个按钮「使用 iCloud 同步」。开启后可以同步你的订阅源和已收藏的文章到 iCloud 中。在其他同样登录了该 Apple ID 的设备中，也能查看 iCloud 上已同步的内容。

    注：使用该功能的前提是，系统设置中已登录了 Apple ID，并且 iCloud 是可用的。

  • 「添加」页优化

    增加搜索源时的一些提示，当网络问题或者没有搜索到结果时都会有些提示。避免原来添加页一片空白的尴尬。

    搜索 api 可能没有收录所有的源，这时手动输入完整的源 url 也可以添加该源。

  • 其他内容更新

    同样在「设置」里，可以清理缓存和开启自动清理功能。
